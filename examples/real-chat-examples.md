# 真实感聊天示例

这些示例是虚构的，用来校准输出风格：短、自然、看上下文、不强行完整、不写成模板作文。

## 朋友礼貌收尾

对方：

```text
我先自己试试 谢啦
```

推荐：

```text
好嘞 有问题再喊我
```

也可以：

```text
行 你先试试 不行我再帮你看
```

要点：

- 对方是在礼貌收尾
- 轻轻接住即可
- 不要继续热情解释

## 夸朋友努力

用户想表达：朋友最近很努力

推荐：

```text
你最近是真的挺能坚持的
```

更自然：

```text
说真的 你这股劲儿挺难得的
```

更熟一点：

```text
你最近做事挺稳的
```

要点：

- 夸努力要具体
- 不要只说“你太厉害了”
- 如果对方容易害羞，夸得轻一点

## 老师鼓励你

对方：

```text
继续保持
```

推荐：

```text
好的老师 我会继续努力的
```

要点：

- 老师场景要稳
- 不需要玩梗
- 不需要过度展开

## 亲密关系里继续夸

短句：

```text
你怎么这么会让人喜欢啊
```

长一点：

```text
你身上有一种很自然的吸引力，不是刻意表现出来的，就是相处久了会越来越觉得舒服，越看越喜欢
```

要点：

- 亲密关系可以更直接
- 语气可以更有情绪
- 不要像写颁奖词

## 夸得太突然，对方没接住

用户：

```text
你最近状态真的挺猛的
```

对方：

```text
啊？
```

推荐：

```text
哈哈哈我刚才说得有点突然 就是觉得你最近挺认真
```

如果对方继续疑惑：

```text
没啥 就是突然感慨一下
```

要点：

- 对方疑惑时，先承认“说得突然”
- 不要继续升级夸奖
- 把本意压成一句普通话

## 朋友说你说话像 AI

对方：

```text
你这话有点像 AI
```

推荐：

```text
哈哈哈确实 我刚才说得太端着了
```

更轻松：

```text
笑死 是有点不像人话了
```

要点：

- 承认表达方式问题
- 不要解释为什么这么说
- 不要继续补充一大段

## 朋友发来一句玩笑

对方：

```text
逮到你了
```

推荐：

```text
抓不到 我溜了
```

要点：

- 接梗即可
- 不要解释
- 不要认真反驳

## 朋友催你去忙自己的事

对方：

```text
行了 快去忙你的项目吧
```

推荐：

```text
行 我去拯救项目现场
```

更稳一点：

```text
行 那我先去忙了
```

要点：

- 顺着话题接
- 不自损
- 不加句号也可以

## 朋友给你起外号

对方：

```text
你真是个小机灵
```

推荐：

```text
那当然 限定版的
```

更轻松：

```text
低调 这都被你发现了
```

要点：

- 如果用户不想自损，就不要顺着贬低自己
- 可以轻轻抬一下自己
- 不要认真辩解

## 用户明确给风格限制

用户要求：

```text
语气轻松一点 不要自嘲
```

推荐输出要遵守：

```text
还行吧 我偶尔也挺靠谱的
```

不建议：

```text
哈哈哈我是废物。
```

要点：

- 用户的风格限制优先级最高
- 不要为了幽默违背限制
