# Gemini / DeepSeek / Kimi / 通义 / 豆包 用法

这些模型都可以直接使用通用提示词。为了稳定输出，建议把任务拆清楚。

## 通用模板

```text
请帮我写中文人情世故话术。

对象：
关系：
场景：
目的：
语气：
字数：

要求：
1. 真诚自然，不油腻。
2. 具体一点，不要只说“太厉害了”。
3. 给我3个版本。
4. 告诉我哪种说法不要用。
```

## 恭维话模板

```text
请帮我写几句恭维话。

我要夸的人：
具体发生了什么：
我想夸的点：
发送场景：
关系远近：
语气：

输出：
- 稳妥版
- 自然版
- 高级一点版
- 不要这么说
```
