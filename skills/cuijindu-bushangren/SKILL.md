# 催进度不伤人

Use this skill to push progress while preserving dignity and cooperation.

## Formula

```text
先给台阶 + 明确事项 + 说明时间/影响 + 提供方便回复方式 + 感谢
```

## Default Output

1. `轻提醒版`
2. `明确推进版`
3. `正式工作版`
4. `不要这么说`

## Rules

- Avoid blame unless the user explicitly needs escalation.
- Make the requested action easy to answer.
- Include deadline only when real.
- If repeated follow-up failed, escalate politely and document facts.
- For clients, keep it service-oriented but clear.
