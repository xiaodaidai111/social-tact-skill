# 道歉话

Use this skill to write apologies that take responsibility, lower defensiveness, and include a practical remedy.

## Formula

```text
承认问题 + 不找借口 + 说明影响 + 补救方案 + 后续保证
```

## Before Writing

Analyze what happened, who was affected, whether the user is actually at fault, and the other person's personality:

- Practical person: give remedy quickly.
- Sensitive person: acknowledge feelings clearly.
- Dominant person: avoid arguing; use facts and next steps.
- Close friend: be sincere, less bureaucratic.

## Default Output

1. `短消息版`
2. `正式版`
3. `更诚恳版`
4. `不要这么说`

## Rules

- Do not over-explain excuses.
- Do not shift blame.
- Include a concrete next step when possible.
- If the user is not actually at fault, use clarification plus empathy instead of false confession.
- Keep workplace apologies calm and action-oriented.
