# 恭维话模板

## 领导 / 上级

重点：尊重、具体、不要像在评价对方。

```text
您刚才把复杂问题讲得特别清楚，我感觉最有帮助的是那几个判断标准，后面我们执行起来会更有方向。
```

```text
这次安排能看出来您提前想得很细，很多容易卡住的地方都被提前处理掉了。
```

## 老师 / 前辈

重点：受益、启发、感谢。

```text
老师您今天讲的那个例子让我一下子理解了，之前我一直停在概念上，现在知道该怎么用了。
```

```text
跟您请教完之后，我感觉思路清楚了很多，尤其是您提醒我的那个角度，特别关键。
```

## 同事 / 同学

重点：自然、具体、轻松。

```text
你这个整理太清楚了，我刚看一遍就知道重点在哪，省了很多沟通成本。
```

```text
你这次反应真的快，而且处理得很稳，换我可能还要多想一会儿。
```

## 客户 / 合作方

重点：专业、判断、配合。

```text
您这个反馈很具体，对我们调整方案特别有帮助，也能看出来您对目标非常清楚。
```

```text
这次沟通效率很高，关键问题都很快对齐了，和您合作确实省心。
```

## 朋友

重点：真诚、轻松、不端着。

```text
你这个想法挺妙的，不是那种硬凑出来的，是真的有点巧思。
```

```text
你身上很难得的一点是，做事有热情，但又不是瞎冲，挺稳的。
```

## 公开评论

重点：短、具体、可公开。

```text
表达很清楚，信息量也够，最喜欢中间那个例子。
```

```text
这个视角很有启发，不是泛泛而谈，能看出来做了很多思考。
```
