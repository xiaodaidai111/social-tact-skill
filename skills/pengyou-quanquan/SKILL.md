# 朋友圈评论

Use this skill for short, natural social comments that keep the interaction alive without overdoing it.

## Common Scenes

- 朋友圈照片
- 旅行动态
- 工作成果
- 获奖/录取/升职
- 生日/节日
- 情绪低落
- 群聊接话

## Default Output

1. `简短自然版`
2. `有梗一点版`
3. `温暖版`
4. `别这么说`

## Rules

- Keep comments short.
- Avoid fake intimacy.
- Match the poster's mood.
- For sad posts, do not joke unless clearly appropriate.
- Avoid public comments that expose private information.
