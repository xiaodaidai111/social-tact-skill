# 上下文与性格分析

写话术前先看人、看关系、看局面。

## 需要判断的信息

- 对象身份：领导、老师、长辈、客户、同事、朋友、同学、伴侣、陌生人。
- 关系远近：熟、不熟、上下级、平辈、第一次见、需要长期维护。
- 场景目的：夸奖、感谢、道歉、拒绝、请人帮忙、催进度、饭局、送礼、评论。
- 对方性格：务实、敏感、强势、内向、爱面子、幽默、急性子、慢热、边界感强。
- 当前上下文：之前聊了什么、对方情绪如何、是否有压力、是否公开场合。
- 用户目的：推进事情、修复关系、表达感谢、留好印象、保持边界、降低尴尬。

## 对症下药

### 务实型

- 少铺垫，多讲具体事情。
- 话术要短、清楚、有行动点。
- 避免过多情绪表达。

### 敏感型

- 多给台阶，少用压迫性表达。
- 先承认对方感受，再说事情。
- 避免“你怎么还没”“你是不是”。

### 强势型

- 尊重对方判断，同时把边界说清。
- 不硬顶，不卑微。
- 用事实、选项和后果组织表达。

### 内向型

- 降低社交压力。
- 避免要求对方立即回应。
- 用“方便的话”“有空时”这类低压力表达。

### 爱面子型

- 保留尊重和余地。
- 私下提醒，公开赞美。
- 避免让对方当场难堪。

### 幽默型

- 可以轻松一点，但正事仍要清楚。
- 玩笑放在正事之后。
- 避免阴阳怪气。

## 输出前检查

- 这句话是否符合关系远近？
- 是否照顾了对方性格？
- 是否达到了用户目的？
- 是否留了台阶？
- 是否有明确边界？
- 是否可以直接复制发送？
