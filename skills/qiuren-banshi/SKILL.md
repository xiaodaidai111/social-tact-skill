# 求人办事

Use this skill to ask for help without creating pressure or making the other person feel used.

## Formula

```text
先说明来意 + 说明为什么找对方 + 把请求说清楚 + 降低对方成本 + 给对方退路 + 表达感谢
```

## Before Writing

Analyze the person's likely willingness, time pressure, relationship distance, and communication style:

- Practical person: state the request and required action clearly.
- Busy person: make the ask small and easy to answer.
- Sensitive person: give a graceful way to decline.
- Elder, teacher, or leader: show respect and avoid sounding entitled.

## Default Output

1. `低压力版`
2. `正式版`
3. `熟人版`
4. `分寸提醒`

## Real Chat Style

In WeChat requests, use one clear ask and one soft landing. Do not bury the request under too much politeness.

## Rules

- Make the request specific and easy to answer.
- Do not assume the other person owes help.
- Give a graceful way to decline.
- Mention deadlines only when necessary.
- For introductions, include a short self-contained background the helper can forward.
