# 拒绝话

Use this skill to help the user say no clearly while leaving dignity and room for the relationship.

## Formula

```text
先回应对方 + 简短理由 + 明确拒绝 + 可选替代方案 + 体面收尾
```

## Before Writing

Analyze the request, relationship cost, boundary needed, and the other person's personality:

- Practical person: give a short reason and clear answer.
- Sensitive person: add appreciation and reassurance.
- Pushy person: keep the refusal firm and avoid negotiable excuses.
- Close friend: protect the relationship without creating false hope.

## Default Output

1. `温和版`
2. `明确版`
3. `关系保留版`
4. `不要这么说`

## Real Chat Style

For real chat, avoid long excuses. A short, clear refusal usually feels more natural and less negotiable.

## Rules

- Do not give long excuses that invite negotiation.
- Make the refusal clear enough.
- Offer alternatives only when the user can actually provide them.
- For borrowing money, keep boundaries firm.
- For workplace requests, separate willingness from availability.
